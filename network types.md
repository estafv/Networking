# Network Types

| Data Unit | Layer | Name | Description |
|-----------|-------|------|-------------|
| WAN | — | Wide Area Network | Connects networks across large distances |
| Data | — | Local Area Network | Connects devices within a local area |
| Data | 5 | Session | Maintains connections and controls communication sessions |
| Segments | 4 | Transport | Service-to-Service communication |
| Packets | 3 | Network | End-to-End communication |
| Frames | 2 | Data Link | Hop-to-Hop communication |
| Bits | 1 | Physical | Transports bits across the physical medium |
