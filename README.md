🌐 Computer Networking

Welcome to my Computer Networking repository! This is where I keep everything I've learned and worked on while studying networking. From the basics of how devices communicate to more advanced topics like routing, subnetting, ect. This repository serves as both my study guide and a portfolio of my progress.

Inside you'll find organized notes, diagrams, labs, assignments, and projects covering a wide range of networking concepts. Topics include the OSI and TCP/IP models, different network types (LAN, WAN, WLAN, VPN, PAN, and more), IP addressing, subnetting, routing, switching, DNS, DHCP, network protocols, wireless networking, and security fundamentals. As I continue learning, I'll keep adding new material and improving existing notes.

This repository isn't just a place to store coursework. It's a way for me to document my learning journey and build a reference I can come back to whenever I need it. Whether I'm reviewing for exams, working on a lab, or learning something new, everything I create will be organized here.
