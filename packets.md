# Packet

A **packet** is a unit of data sent across a network.

A packet contains:
- **Header**  Control information used for delivery such as source/destination addresses and protocol information.
- **Payload**  The actual data being transmitted such as messages, images, or files.

Protocols like **TCP** and **UDP** add their own headers to help manage communication.
