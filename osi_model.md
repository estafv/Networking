# The OSI Model

The OSI (Open Systems Interconnection) is a model that explains how computers communicate over a network using 7 layers, each with a specific job.

ALL PEOPLE SEEM TO NEED DATA PROCESSING

| Data | Layer | Functions |
| --- | --- | --- |
| Data | 7  Application | Human computer interaction layer, where applications can access network services |
| Data | 6  Presentation | Ensures that data is in a usable format and handles data encryption |
| Data | 5  Session | Maintains connections and controls communication sessions |
| Segments | 4  Transport | Service to Service |
| Packets | 3  Network | End to End |
| Frames | 2  Data Link | Hop to Hop  |
| Bits | 1  Physical | Transporting Bits |
