# The OSI Model

The Open Systems Interconnection (OSI) model describes seven layers that computer systems use to communicate over a network.

| Data | Layer | Functions |
| --- | --- | --- |
| Data | 7 - Application | Human-computer interaction layer, where applications can access network services |
| Data | 6 - Presentation | Ensures that data is in a usable format and handles data encryption |
| Data | 5 - Session | Maintains connections and controls communication sessions |
| Segments | 4 - Transport | Transmits data using protocols including TCP and UDP |
| Packets | 3 - Network | Determines the path that data takes across a network |
| Frames | 2 - Data Link | Defines how data is formatted and handles MAC addresses |
| Bits | 1 - Physical | Transmits raw bits over the physical medium |
